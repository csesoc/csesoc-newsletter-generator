COMP9447
SecEdu would like to let you all know that COMP9447 is taking expressions of interest for 22T2, 22T3, and even 2023 if you're thinking that far ahead. COMP9447 is a Security Engineering Workshop. It's project-based and you'll be put in teams, paired with AWS mentors to build a security solution. You'll also be using AWS products - a great experience to learn how to use cloud computing services and develop your cloud computing skills altogether.

This is a really practical course, and if you're considering pursuing either cloud computing or cyber security as a career path, this will definitely get you a shoo-in.

The expression of interest form is here: https://forms.office.com/r/nj61ykenV1

Make sure to read the requirements carefully - COMP6441 (or COMP6481) is a prereq, and the course will take up approx 15 hours/week.

Any questions, email cs9447@cse.unsw.edu.au or check out our website here (https://sec.edu.au/comp9447).
